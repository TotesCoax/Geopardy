Initial Setup:
Google sheets link for template: https://docs.google.com/spreadsheets/d/1Y1cFkHU2LjpyEphxHNiNDZqdBl2SwX1IQATNtCvbpPY/edit?usp=sharing
Fill this in and export/save as a csv for importing into the page.
You use the second tab (answer sheet) during the game for easier reference.

Controls:

Clicking on the Geopardy Banner opens up an admin panel for changing the names

Double-clicking on teach point value will display the question fullscreen
Double-clicking again will remove it from fullscreen and mark it as answered.

For assigning points, you can click into the point value and either: enter a new amount directly or use the up and down arrows to increment by 100.
